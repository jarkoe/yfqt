男孩子都懂的填空题	惟有在坎坷时才懂，愿拉你一把的人何其少。惟有在最穷时才懂，再好的情绪也难敌实际，人不贪钱却都怕刻苦。惟有在长久生存里才懂，放荡易忍受难，大众能爱你却少人愿忍你。以是，这世上最要保护的是这三种人：锦上添花的伙伴、愿陪你流过艰难的女子、句句都忍你的男子。
“我成了比祥林嫂还典型的祥林嫂，我流亡自己的语言，不停地寻找倾诉的对象，我希望有一只高足容器把我的言语之雨一一装殓妥当。我的话语如汪洋恣肆的洪水，一只高足容器溢满为患，我又寻找不同的语言停靠支点，我抓住一切，迫不及待，滔滔不绝地诉说，强行时间停驻在某一点上……”
夜色阑珊，月华如练，离别的一幕，若一层迷蒙的青霭，笼罩于我的眉间。风卷起素笺，想与不想，念与不念，你都在心里，梦里，从未走远。望断时光的彼岸，穿越岁月的烟岚，你的城，幽幽跳动，心香一瓣。我期待的眼神，在一片月光中流连忘返。
　　这是大多数人的看法。我却不以为然。我以为这种认识颠倒了逻辑关系。世道险恶，缘于人心险恶。世道叵测，缘于人心叵测。世道并非超乎个人之上存在的天道，而是由每一个具体的人的“人道”组成的。假如我们每一个人都洁身自好，假如我们每一个人都让内心的灯盏亮着，世道就不会像我们想象的那么黑了。
　　没有钱上病院，弟弟朝不保夕。谁人冬天，妈妈垂泪坐在床上，弟弟躺在她的臂弯，胸口打开，不许穿衣物不许盖被卧，由于遇上什么何处仍旧溃乱的皮肤城市黏上。爸爸四处找猫骨头，磨成粉，用香油调，而后用羽毛轻轻给弟弟搽上。那是民间治烫伤的丹方，尽管有没有效，总要试一试。
